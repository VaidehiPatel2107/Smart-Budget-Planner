# Smart Budget Planner (OOP Based)

## Description
A C++ command-line application that helps users manage their budget using category-wise limits and expense tracking.

## Features
- Add categories
- Set spending limits
- Add expenses
- Warning when limit exceeds
- Budget summary report

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Encapsulation
- Constructors
- Vectors (STL)
- Menu-driven programming

## How to Run
1. Compile using a C++ compiler
2. Run the program
3. Follow the menu options# Smart-Budget-Planner
